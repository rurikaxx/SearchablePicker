# SearchablePicker
### Searchbar+Picker的應用
### 個人覺得這個UX概念挺不錯的
### Picker改寫於對岸同胞寫的CycleWheelView[滾動選擇器]
### 由於Picker繼承ListView
### 因此只要改寫Adapter List內容就可以是各種View
### CycleWheelView出處:http://www.bkjia.com/Androidjc/1031796.html
### 本人主要就是加上放大縮小動畫 & 新增SearchBar & 修正一些小issue & 新增Picker的OnItemClick Event處理
